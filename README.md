# Insurance loss estimation

Rest API for training and evaluation insurance loss estimation models.

## API Launch

In the project root directory run:

```bash
make up-test
```
to launch the application in test mode. By default, the application will be available at:

http://localhost:8000/api/v1/docs.

Refer to the [Makefile](Makefile) for other launch commands and [test.env](test.env) for configuration variables.

## Tests

To run tests, execute from the project root directory:

```bash
make up-test
make run-tests
```
